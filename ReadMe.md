# このソフトウェアについて #

DrawMultipleMouse20160706は私個人が学習目的で作成したソフトウェアです。

複数のマウスポインタを描画します。

# 開発環境 #

* Windows XP SP3
* C++/Win32
* Visual C++ 2010 Express
